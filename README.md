究極の炭火焼き、ここに極まる。伝統と炎が織りなす、厳選した炭とこだわりの肉が奏でる至高の極み。

## 📋 プロジェクト概要

これは「鳥とん」の公式ウェブサイトです。Stitchで生成されたUIをベースに構築された静的サイトです。

### ページ構成

- **ホーム（index.html）** - トップページ
- **お品書き（menu.html）** - メニュー一覧
- **こだわり（philosophy.html）** - 私たちのこだわり
- **店舗情報（store.html）** - アクセス・営業時間
- **お問い合わせ（contact.html）** - お問い合わせフォーム

## 🚀 技術スタック

- **HTML5** - マークアップ
- **Tailwind CSS** - スタイリング（CDN版）
- **Google Fonts** - フォント
- **Material Symbols** - アイコン

## 🎨 デザイン

- ダークモードベースのデザイン
- 黒（#131313）を基調とした高級感のあるカラースキーム
- プライマリーカラー：#ffb3b1（柔らかいピンク）
- サブカラーアクセント：#ffba3f（ゴールド）

## 📦 セットアップ

1. リポジトリをクローン

```bash
git clone https://github.com/sushihut35-code/toritonHP.git
cd toritonHP
```

2. ローカルサーバーで起動

```bash
# Python 3
python -m http.server 8000

# Node.js (http-server使用)
npx http-server
```

3. ブラウザで `http://localhost:8000` にアクセス
